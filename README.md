# Task 5: Logistic Regression

## Project Overview

In this task, I worked on implementing a Logistic Regression model for binary classification using the Breast Cancer dataset. The goal was to understand how classification models work and evaluate their performance.

##  Tools Used

* Python
* NumPy
* Pandas
* Scikit-learn

##  Dataset

I used the Breast Cancer dataset available in sklearn.

* Target values:

  * 0 → Malignant (Cancerous)
  * 1 → Benign (Non-cancerous)

---

##  Steps Performed

1. Loaded the dataset using sklearn
2. Converted the dataset into a Pandas DataFrame
3. Separated features (X) and target (y)
4. Split the dataset into training and testing sets
5. Created a Logistic Regression model
6. Trained the model using training data
7. Predicted results using test data
8. Evaluated the model using accuracy and classification report

---

## Results

* The model achieved good accuracy on the test data
* Generated a classification report including:

  * Precision
  * Recall
  * F1-score

* Understanding of classification problems
* Logistic Regression algorithm
* Sigmoid function
* Model training and testing
* Performance evaluation

This task helped me understand the fundamentals of binary classification and how Logistic Regression works in real-world datasets.
